# saarthi-ai
SAARTHI AI is a voice-first emotion-aware learning and productivity mentor that guides users step-by-step through technology tasks using real-time conversational assistance, context memory, and multilingual support across smartphones and feature phones.
## Prototype Modules

The prototype includes a dictionary-based explanation module that helps learners understand programming concepts instantly during guided learning sessions.

This module enables SAARTHI AI to:
- Provide quick definitions of technical terms
- Explain programming concepts with examples
- Support learners while they are actively working on tasks.
